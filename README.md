## 🎧 خدمة العملاء

### برومبت 1: الرد على الشكاوى باحترافية

```
أنت موظف خدمة عملاء محترف في شركة [اسم الشركة] في [المدينة/الدولة].
تلقيت الشكوى التالية من العميل: "[نص الشكوى]"

قم بـ:
1. الاعتذار بصدق وتعاطف دون قبول المسؤولية القانونية
2. شكر العميل على تواصله
3. تقديم حل عملي واضح مع جدول زمني
4. تقديم تعويض مناسب إن لزم
5. إنهاء الرسالة بطريقة تُعزز الثقة

اكتب الرد بلغة عربية فصيحة مهذبة، لا تتجاوز 150 كلمة.
```

---

### برومبت 2: تصعيد وتصنيف شكاوى العملاء

```
لديك نظام دعم عملاء. استلمت الرسائل التالية من العملاء:
[الصق الرسائل هنا]

صنّف كل رسالة حسب:
- الأولوية: (عاجل / متوسط / منخفض)
- النوع: (شكوى / استفسار / طلب استرداد / اقتراح)
- القسم المسؤول: (تقني / مالي / توصيل / جودة المنتج)
- اقتراح الرد الأولي (جملة واحدة)

أخرج النتيجة في جدول منظم.
```

---

### برومبت 3: قاعدة معرفة للأسئلة الشائعة

```
أنت مسؤول عن إنشاء قاعدة معرفة لـ [اسم الشركة] في قطاع [القطاع].

بناءً على هذه القائمة من شكاوى العملاء الشائعة:
[الصق الشكاوى هنا]

اكتب 10 إجابات جاهزة بالعربية لأكثر الأسئلة تكراراً، مع:
- صياغة واضحة ومباشرة
- نبرة ودية ومهنية
- خطوات عملية قابلة للتنفيذ
- إضافة جملة ختامية تدعو للتواصل المباشر
```

## 📍 جوجل بزنس (Google Business Profile)

### برومبت 4: كتابة وصف احترافي للنشاط التجاري

```
اكتب وصفاً احترافياً لنشاط تجاري بالعربية لرفعه على Google Business Profile.

معلومات النشاط:
- الاسم: [اسم النشاط]
- القطاع: [المجال]
- الموقع: [المدينة، الدولة]
- خدماتنا الرئيسية: [الخدمات]
- ما يميزنا: [نقاط التميز]

المتطلبات:
- الطول: 250-750 حرف
- ضمّن 3-5 كلمات مفتاحية طبيعياً
- ابدأ بجملة قوية تجذب الانتباه
- اختم بدعوة واضحة للتصرف (CTA)
- لا تستخدم كلمات مبالغ فيها أو وعود غير واقعية
```

---

### برومبت 5: الرد على تقييمات جوجل

```
أنت مدير [نشاط تجاري] في [المدينة]. اكتب ردوداً على التقييمات التالية:

التقييم 1 (5 نجوم): "[نص التقييم الإيجابي]"
التقييم 2 (2 نجوم): "[نص التقييم السلبي]"
التقييم 3 (4 نجوم): "[نص التقييم المتوسط]"

لكل رد:
- اجعله فريداً (لا تكرر نفس القالب)
- اذكر اسم العميل إن وُجد
- للإيجابية: شكر حار + دعوة للعودة
- للسلبية: اعتذار + حل + دعوة للتواصل الخاص
- الطول المثالي: 50-80 كلمة
- نبرة: احترافية وودية
```

## 🏥 العيادات الطبية

### برومبت 6: تذكير المرضى بالمواعيد

```
أنت مساعد إداري في [اسم العيادة/المستشفى]. اكتب رسالة WhatsApp لتذكير المرضى بمواعيدهم.

تفاصيل الموعد:
- اسم المريض: [الاسم]
- الطبيب: [د. الاسم] - [التخصص]
- التاريخ والوقت: [التاريخ] الساعة [الوقت]
- نوع الزيارة: (متابعة / كشف أول / إجراء)

المتطلبات:
- رسالة قصيرة لا تتجاوز 5 أسطر
- نبرة محترمة ودافئة
- أضف تعليمات ما قبل الزيارة إن كانت ضرورية
- أنهِ بطلب التأكيد (رد بـ "نعم" للتأكيد)
- أضف رقم للتواصل في حال الإلغاء
```

---

### برومبت 7: محتوى توعوي طبي لوسائل التواصل

```
أنت طبيب [التخصص] في [المدينة]. اكتب 5 منشورات توعوية قصيرة لانستغرام/تويتر عن [موضوع طبي].

لكل منشور:
- الطول: 100-150 كلمة
- ابدأ بحقيقة مفاجئة أو سؤال يثير الاهتمام
- قدم معلومة طبية دقيقة ومبسّطة
- أضف نصيحة عملية قابلة للتطبيق
- اختم بـ hashtags عربية وإنجليزية مناسبة (5-7 وسوم)
- نبرة: علمية لكن مفهومة للعموم
- لا تتضمن أي دعاية مباشرة للعيادة
```

## 🍽️ المطاعم

### برومبت 8: كتابة قائمة طعام جذابة

```
أنت كاتب محتوى متخصص في صناعة الأغذية. اكتب أوصافاً جذابة لقائمة الطعام لـ [اسم المطعم].

الأطباق:
1. [اسم الطبق]: المكونات الرئيسية: [المكونات]
2. [اسم الطبق]: المكونات الرئيسية: [المكونات]
3. [اسم الطبق]: المكونات الرئيسية: [المكونات]

لكل طبق:
- وصف شهي يثير الشهية (40-60 كلمة)
- أبرز المكونات الفريدة أو المحلية
- اذكر الحساسيات الغذائية إن وُجدت (جلوتين، مكسرات، ألبان)
- استخدم كلمات حسية (مقرمش، دسم، طازج، بطيء الطهي...)
```

---

### برومبت 9: حملة تسويقية لعروض المطعم

```
اكتب حملة تسويقية متكاملة على وسائل التواصل لعرض [اسم العرض] في [اسم المطعم].

تفاصيل العرض:
- الأطباق المشمولة: [الأطباق]
- مدة العرض: [من - إلى]
- السعر/الخصم: [التفاصيل]
- الجمهور المستهدف: [الفئة]

المطلوب:
1. منشور انستغرام (مع كابشن + هاشتاقات)
2. ستوري انستغرام (جملتان + CTA)
3. رسالة WhatsApp للعملاء المسجلين
4. تغريدة X (تويتر)

أسلوب: حماسي، يثير الشهية، يخلق إلحاحية (FOMO).
```

## 👥 الموارد البشرية

### برومبت 10: كتابة إعلانات الوظائف

```
أنت مدير موارد بشرية في [اسم الشركة]. اكتب إعلان وظيفة احترافي وجذاب.

المعلومات:
- المسمى الوظيفي: [المسمى]
- القسم: [القسم]
- نوع العمل: (دوام كامل / جزئي / عن بُعد / هجين)
- المدينة: [المدينة]
- المتطلبات الأساسية: [المتطلبات]
- المهام الرئيسية: [المهام]
- المميزات: [المزايا]

المطلوب:
- عنوان جذاب يثير الاهتمام
- وصف الشركة والبيئة (50 كلمة)
- المهام والمسؤوليات (قائمة نقطية)
- المتطلبات (إلزامية + مُفضَّلة)
- ما يميز العمل معنا
- طريقة التقديم وآخر موعد
اجعله يعكس ثقافة الشركة ويجذب المواهب المناسبة.
```

---

### برومبت 11: تقييم الأداء الوظيفي

```
أنت مدير قسم. اكتب تقرير تقييم أداء سنوي للموظف التالي:

معلومات الموظف:
- الاسم: [الاسم]
- المسمى: [المسمى]
- فترة التقييم: [الفترة]
- الإنجازات الموثقة: [الإنجازات]
- نقاط التحسين الملاحظة: [الملاحظات]
- الأهداف المحددة مسبقاً: [الأهداف]
- نسبة تحقق الأهداف: [النسبة]%

المطلوب:
- ملخص تنفيذي (100 كلمة)
- تقييم كل محور (القيادة، التعاون، الإنتاجية، الالتزام) من 5
- أبرز الإنجازات
- مجالات التطوير مع خطة عمل
- أهداف العام القادم (SMART)
نبرة: بنّاءة ومحفِّزة.
```

## 📱 وسائل التواصل الاجتماعي

### برومبت 12: خطة محتوى شهرية

```
أنت مدير وسائل تواصل اجتماعي. ضع خطة محتوى لشهر كامل لـ [نوع الحساب/الشركة] في قطاع [القطاع].

المعلومات:
- المنصات: [انستغرام / تويتر X / لينكدإن / تيك توك]
- الجمهور المستهدف: [وصف الجمهور]
- الأهداف: [التوعية / المبيعات / التفاعل]
- التكرار المطلوب: [عدد المنشورات أسبوعياً]
- المناسبات في الشهر: [المناسبات]

المطلوب:
- جدول أسبوعي (30 يوم)
- لكل يوم: نوع المحتوى + الفكرة الرئيسية + المنصة
- توزيع: 40% تعليمي، 30% ترفيهي، 20% ترويجي، 10% تفاعلي
- قائمة بـ 10 هاشتاقات مناسبة للقطاع
```

---

### برومبت 13: تحليل المنافسين على السوشيال ميديا

```
حلِّل الحضور الرقمي للمنافسين التاليين في قطاع [القطاع]:

المنافسون:
1. [اسم المنافس 1] - [رابط أو وصف]
2. [اسم المنافس 2] - [رابط أو وصف]
3. [اسم المنافس 3] - [رابط أو وصف]

المطلوب تحليله:
- نوع المحتوى الأكثر تفاعلاً
- التكرار والتوقيت
- نبرة الصوت والهوية البصرية
- نقاط القوة والضعف لكل منافس
- الفرص الغير مستغلة في السوق
- 5 توصيات استراتيجية لتميّزك عنهم

أخرج النتائج في جدول مقارنة + ملخص تنفيذي.
```

### برومبت 14: كتابة بيوغرافي احترافي للحسابات

```
اكتب بيوغرافي (Bio) احترافي لحسابات التواصل الاجتماعي لـ [الاسم/الشركة].

المعلومات:
- الاسم: [الاسم]
- المجال: [المجال/التخصص]
- الإنجازات البارزة: [الإنجازات]
- القيم الجوهرية: [القيم]
- الجمهور المستهدف: [الجمهور]
- CTA المطلوب: [ما تريد أن يفعله الزوار]

اكتب نسخة مُحسَّنة لكل منصة:
- انستغرام: 150 حرف
- تويتر X: 160 حرف
- لينكدإن: 300 كلمة (ملف احترافي كامل)
- تيك توك: 80 حرف + إيموجيات مناسبة
```

### نصائح للحصول على أفضل النتائج
1. **كن محدداً** — كلما أضفت تفاصيل أكثر، كان الناتج أفضل
2. **استخدم أمثلة** — أضف مثالاً على ما تريد في نهاية البرومبت
3. **كرر وحسّن** — اطلب من الأداة تحسين ناتجها: "حسِّن الفقرة الثانية"
4. **حدد الجمهور** — وصف من سيقرأ النص يُغيّر النبرة تلقائياً
5. **اضبط الطول** — حدد عدد الكلمات المطلوبة دائماً
